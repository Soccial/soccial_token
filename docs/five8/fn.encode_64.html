pub fn encode_64(bytes: &[u8; 64], out: &mut [u8; 88]) -> u8
Expand description

Encode a 64-byte array.

Mutates the provided out array and returns a u8 len which indicates how many bytes of the out array were actually written to. The remaining bytes are unchanged, hence the result we care about after calling the function is out[..len as usize].
